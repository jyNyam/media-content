# 🎥 Digital Media Portfolio | 디지털 콘텐츠 포트폴리오

 안녕하세요. 저는 **교육 및 정보 전달을 위한 디지털 콘텐츠**를 기획하고, 편집하고, 사용자 경험을 고려해 설계하는 콘텐츠 제작자입니다.
